linux后台运行

发表于 2018-10-29

*xshell退出保持后台服务运行的方法

Linux后台启动了一个服务，但是退出命令终端后或者退出xshell后，服务就关闭了，要想保持后台服务一直启动，可以使用下面的命令来启动服务

nohup java -jar app.jar –server.port=8088 &

其中nohup和&中间是启动服务的命令(即正常启动的命令, 上面的例子是启动一个spring boot 服务)
